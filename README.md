# DC Motor with Series Excitation (ДПТ ПВ)

Модель двигателя постоянного тока с последовательным возбуждением в MATLAB/Simulink. Лабораторная работа по курсу "Нелинейные системы автоматического управления".

## 📁 Структура проекта
📦 DC_motor
## 📂 Описание папок

| Папка | Назначение |
|-------|------------|
| `models` | Содержит Simulink модель двигателя |
| `Const` | Скрипты с константами и параметрами |
| `Tests` | Модульные тесты для проверки модели |
| `Shortcuts` | Утилиты для запуска и очистки |
| `docs` | Документация и отчёты |
| `resources` | Изображения и дополнительные файлы |
| `slprj` | Временные файлы (можно игнорировать) |

## ⚙️ Основные файлы

- **`DC_motor.slx`** — главная модель для симуляции
- **`Initial_const.m`** — задаёт параметры двигателя
- **`DCMotorTest.m`** — автоматическая проверка модели
- **`Run_tests.m`** — запуск всех тестов одной командой
- **`Initialization.m`** — подготовка workspace
- **`Clear_trash.m`** — удаление временных файлов

## 🚀 Быстрый старт

1. Откройте MATLAB
2. Перейдите в папку проекта
3. Дважды кликните `DC_motor.prj`
4. Запустите `Initialization.m`
5. Откройте `models/DC_motor.slx` и нажмите Run
